这是hcz写的atcoder-better安装教程

注意

本版本为hcz特制,仅限南渝信竞班自用

第一步,安装篡改猴(如已经安装请跳过)

edge

1.打开插件市场或点我

2.搜索篡改猴并安装

chrome

篡改猴 chrome edge 等下载

视频教程
下载视频教程

下载压缩包后解压缩

在设置中找到拓展程序

打开开发者模式,选择已解压缩的拓展->选择已经解压的文件夹,注意是最里面的一层

firefox

最简单,点链接安装
篡改猴 firefox

第二步,安装脚本,点击下方按钮

Atcoder better
codefouce better

点击安装即可(点击后要等待一会,初次安装进入atcoder需要等待一会)
